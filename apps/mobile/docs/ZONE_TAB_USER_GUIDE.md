## Getting Started

### Accessing the Zone Tab

1. Open the app and go to **Impact Hub** (bottom navigation)
2. Tap the **Zone** tab at the top
3. If you have events today, you'll see them listed

### No Events Showing?

The Zone tab only shows events you're **registered for** that are happening **today**. If you don't see any events:

- Make sure you've registered for an event
- Check that the event is scheduled for today
- Try refreshing by pulling down on the screen

---

## Check-In & Check-Out

### How to Check In

1. When you arrive at an event venue, open the Zone tab
2. Find your event card
3. Tap the **Check In** button
4. You're now checked in! Other attendees can see you're at the event

### How to Check Out

1. On your checked-in event, tap **Check Out**
2. You'll be marked as no longer at the event

### Why Check In?

- **Networking**: Other attendees can discover you in the "Attendee Radar"
- **Engagement**: Participate in live polls and activities
- **Statistics**: Event organizers can see attendance

---

## Live & Upcoming Sessions

### Live Sessions

Sessions happening right now appear at the top with a **LIVE** badge:

- **Red pulsing indicator** = Currently live
- Shows speaker name, room location, and time remaining
- Tap to see full session details

### Upcoming Sessions

Sessions starting soon are listed below live sessions:

- Shows start time and countdown
- Helps you plan your schedule
- Tap any session for more details

---

## Attendee Radar

The **Networking Radar** shows other attendees currently checked into the same event:

- See profile photos and names
- View headlines/titles to find relevant connections
- **Green dot** = Currently online
- **Shared interests** shown as tags
- Tap a profile to view details and connect

---

## Live Polls

Event organizers can run live polls during sessions. When a poll is active:

1. You'll see the poll card in the Zone tab
2. Read the question and options
3. Tap your preferred option to vote
4. See real-time vote counts update
5. Results show as percentage bars

### Poll Rules

- You can only vote once per poll
- Some polls have time limits
- Your vote is anonymous to other attendees

## Troubleshooting

### "No events for today"

- Confirm you're registered for an event happening today
- Pull down to refresh the Zone tab
- Check your internet connection

### Can't check in

- Make sure you're logged in to your account
- Verify you're registered for the event
- Try closing and reopening the app

### Poll won't accept my vote

- Check if the poll has expired
- Ensure you haven't already voted
- Refresh the page and try again

### Can't see other attendees

- Other attendees must be checked in too
- The attendee radar refreshes periodically
- Try pulling down to refresh

### Sessions not showing

- Sessions must be scheduled for today
- The event must have sessions configured
- Contact the event organizer if you think there's an error

---

## Privacy & Data

- **Check-in status**: Visible to other attendees at the same event
- **Profile info**: Only what you've made public
- **Poll votes**: Anonymous (organizers see totals, not individual votes)
- **Location**: Only stored if you check in with location enabled
